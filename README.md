# S-CG-MoLE
This repository contains R functions used to fit the semi-parametric contaminated Gaussian mixture of regressions model proposed in the article:

_Mambondimumwe, P., Skhosana, S.B. and Rad, N.N., 2026. Robust semi-parametric mixtures of linear experts using the contaminated Gaussian distribution. arXiv preprint arXiv:2601.12425._

Available through the link: https://doi.org/10.48550/arXiv.2601.12425
